# 🎤 Konuşuyorum

Konuşamayan çocuklar için sesli iletişim uygulaması (AAC - Augmentative and Alternative Communication).

## Özellikler

- ✅ Kişi/Kategori/Kelime yönetimi
- ✅ Web Speech API ile sesli okuma (Türkçe)
- ✅ Fotoğraf yükleme desteği
- ✅ PWA (Ana ekrana ekleme)
- ✅ Çocuksu, renkli tasarım
- ✅ Tüm veriler cihazda saklanır (localStorage)

## Kullanım

1. Yeni kişi ekleyin
2. Kategoriler oluşturun
3. Kelimeler/cümleler ekleyin
4. Kelimeye dokunarak sesli oku

## Teknik

- HTML5 + CSS3 + JavaScript
- localStorage (veri depolama)
- Web Speech API (sesli okuma)
- Service Worker (offline destek)
- PWA (ana ekrana ekleme)
